# ECE341-code
This contains all the labs performed for the Microcontrollers course at University of Idaho. <br/><br/>
Every main() should be contained within the source file named after the project (i.e. "Project_#.c"). <br/><br/>
These labs were performed on a Cerebot MX7cK board (based on the Microchip
PIC32MX795F512L, and member of the 32-bit PIC32 microcontroller family).
